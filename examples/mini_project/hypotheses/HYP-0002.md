## Hypothesis

A simple recovery-quality rubric plus an unstructured-note baseline can make the next resumption trial interpretable.

## Prediction

If structured cards help recovery, the structured-card condition should preserve active item, next action, evidence boundary, and branch provenance at least as well as an unstructured-note baseline.

## Refutation Or Weakening

The hypothesis weakens if the rubric cannot be applied consistently, the baseline is not comparable, or the structured condition does not improve any rubric item.

## Minimum Experiment

One paired mock continuation using the same source scenario, comparing structured cards with an unstructured one-page note.

## Required Metric/Baseline Readiness

- Metric: Recovery Quality Score with explicit rubric anchors.
- Baseline: Unstructured note created from the same scenario.
- Threshold: Structured condition must beat baseline on the primary score before any comparative claim is strengthened.
- Ambiguous result: If scores tie or raters disagree, revise the rubric instead of strengthening `CLM-0001`.
