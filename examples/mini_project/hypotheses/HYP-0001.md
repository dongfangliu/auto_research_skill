## Hypothesis

If an interrupted research task has a `STATE.md` file and linked cards, an agent can resume with fewer missing-context errors than from an unstructured note.

## Prediction

The agent should identify the active item, branch relationship, next action, and evidence limitation without asking the user to restate the project.

## Falsification Condition

If the agent misses active items, overwrites old decisions, or strengthens unsupported claims, the workflow is insufficient.

## Background Evidence

Linked to `IDEA-0001`, `LQ-0001`, and the placeholder literature note.

## Minimum Experiment

Create a small interrupted project and ask an agent to continue from `STATE.md` and linked cards.

## Competing Hypotheses

- A shorter plain summary might perform equally well.
- The main benefit may be disciplined documentation, not card structure.

## PI Review

Proceed as a weak mock experiment only. Do not claim general effectiveness.
